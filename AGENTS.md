## Project Summary
SquaerdsSoft Home is a premium, minimalist hub for the SquaerdsSoft ecosystem. It serves as the central landing page connecting WYRM AI, LovorisMc, and SquaerdsSoft Docs. The design is ultra-clean, white-themed, and features a continuous, ethereal motion aesthetic.

## Tech Stack
- Next.js 15+ (App Router)
- Tailwind CSS 4
- Framer Motion
- Lucide Icons
- React 19

## Architecture
- `src/app/`: Core routing and pages
- `src/components/`: Reusable UI components including ecosystem-specific elements
- `src/lib/`: Utility functions and shared logic

## User Preferences
- Clean white theme (beyaz temalı)
- Wallpaper-like continuous flow (aynı wallpaper gibi devam ediyor)
- High-quality "V" aesthetic (premium and minimalist)
- Central hub for SquaerdsSoft projects (WYRM, LovorisMc, Docs)

## Project Guidelines
- Maintain an "ethereal" white-on-white or white-on-grey aesthetic
- Use large typography and generous whitespace
- Integrate visual previews or representations of linked SquaerdsSoft sites
- No unnecessary comments

## Common Patterns
- Floating particle backgrounds
- Bento-style or clean card grids for project links
- Smooth, physics-based animations (Framer Motion)
